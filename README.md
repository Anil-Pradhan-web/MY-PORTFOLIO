# MY-PORTFOLIO

## Anil Pradhan | Backend Architect & Full-Stack Developer

A production-grade, highly optimized personal portfolio built with a modern tech stack and cyberpunk aesthetic.

### 🚀 Tech Stack
- **Framework:** Next.js 16.2.0 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS V4 + Glassmorphism / Neon glow effects
- **Animations:** Framer Motion
- **Icons:** React Icons & Lucide React
- **Deployment:** Vercel

### 🎯 Features
- Dark aesthetic with dynamic glowing cyan/neon highlights
- Highly responsive grid design optimized for mobile and desktop
- Specialized ASCII Terminal component displaying live stats
- Backend-less working contact form using FormSubmit
- Interactive unified project cards and skills display


*Built from ground up with scalability and modern UI logic in mind by Anil Pradhan.*
